# Kafka Spark Streaming

# Quickstart
## Running Docker Compose

To run docker compose simply run the following command:

```
cd kafka-spark-streaming-docker
docker-compose up -d
```

To run simulation:

```
cd streamingProje
python simulator.py
```

To get msg from kafka:

```
cd streamingProje
python get_msg_kafka.py
```
